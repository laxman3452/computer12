Data Definition & Data Manipulation Language

Class 12 Computer Science - DBMS Chapter

🧱 DDL (Data Definition Language)

DDL is a subset of SQL used to define, modify, and remove database structures such as tables, schemas, and indexes. It does NOT deal with actual data inside tables.

💻 DDL Syntax with Explanation

1️⃣ CREATE TABLE

CREATE TABLE Student (
  id INT,
  name VARCHAR(50),
  age INT
);

✔ Creates a new table named Student in database.

2️⃣ ALTER TABLE

ALTER TABLE Student ADD email VARCHAR(100);

✔ Adds a new column "email" to existing table.

3️⃣ DROP TABLE

DROP TABLE Student;

✔ Permanently deletes table structure + data.

4️⃣ TRUNCATE TABLE

TRUNCATE TABLE Student;

✔ Deletes all records but keeps table structure intact.

🧾 DML (Data Manipulation Language)

DML is used to manipulate data stored inside database tables.

📌 Commands:

  • INSERT → Add new records
  • UPDATE → Modify existing data
  • DELETE → Remove specific records
  • SELECT → Retrieve data

💻 Syntax Examples:

INSERT INTO Student VALUES (1, 'Ram', 18);
  
UPDATE Student SET age = 19 WHERE id = 1;
  
DELETE FROM Student WHERE id = 1;
  
SELECT * FROM Student;

📤 Output Example:

+----+------+-----+
| id | name | age |
+----+------+-----+
| 1  | Ram  | 19  |
+----+------+-----+
