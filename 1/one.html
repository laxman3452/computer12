1.1 Introduction

Data: Raw facts without meaning (Ram, 20, Kathmandu)
Information: Processed data with meaning
Database: Organized collection of data
DBMS: Software to manage database
Database System: DB + DBMS + Users + Hardware

1.2 Advantages of DBMS

  • Reduces data redundancy
  • Improves security
  • Easy access and retrieval
  • Data consistency
  • Multi-user support
  • Backup & recovery
  • Data sharing
